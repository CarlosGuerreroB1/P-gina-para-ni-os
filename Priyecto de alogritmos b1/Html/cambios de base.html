Cambio de Bases

Cambiar de base significa representar un número usando diferentes dígitos. Por ejemplo, cambiar de base 10 (decimal) a base 2 (binaria) implica representar un número usando solo los dígitos 0 y 1 en lugar de 0 al 9.

  • Base 10 a base 2
  • Base 10 a base 8
  • Base 10 a base 16
